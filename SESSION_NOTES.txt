PARAMETERS CLARIFIED
---------------------
BATT_FS_CRT_ACT=6 = Loiter to QLand (NOT terminate — corrected
after user pointed out the error).

Carrier Pilot redundancy controller = 915MHz (not 868MHz — corrected
after user pointed out).

FPV relay chain ground→carrier hop = 868MHz.
FPV relay carrier→FPV hop = 2.4GHz.

RF chain has two separate diagrams in manual:
1. FPV relay chain (868→2.4GHz)
2. Carrier Pilot redundancy (915MHz)

All carrier systems powered from single 6S battery — nothing separate.
DBR4 = the RX on carrier (receives 868MHz).
Carrier Nomad = the relay TX on carrier (transmits 2.4GHz to FPV).


FEATURES ADDED TO index.html
------------------------------
1. Preflight timer
   - START/STOP in checklist header
   - MM:SS display, amber after 20min
   - On stop: prompt to archive with notes
   - Archive in localStorage (vtol_pf_archive)
   - Archive table shown below checklist

2. Battery log (.tlog auto-read)
   - File input for .tlog binary files
   - parseTlog() / mavlinkExtractBattery() — pure JS MAVLink v1/v2
     binary parser, no server needed
   - Extracts: start V, min V, end V, flight time → pre-fills form

3. Battery log (MP live fetch)
   - "MP" button next to each voltage field
   - Fetches from http://localhost:20199/mavlink/SYS_STATUS
   - Parses voltage_battery (mV → V)
   - Works only on laptop running Mission Planner with REST API enabled
   - Fails gracefully with instructions if not available

4. Weather tab (Open-Meteo free API)
   - Lat/lon input or browser geolocation
   - Shows wind, gusts, temperature
   - Go/no-go based on T2 wind limits
